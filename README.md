#MyPaintApp
実験で作った図形ペイントアプリ

## Description
GUIウインドウ上で図形をペイントし、選択した図形に何かしらの効果を加えるアプリケーション。2015年前期の実験で作成。
描いた図形を並べたり、アニメーションさせたりできる。
main関数はMyApplication3.javaに記述。

## Requirement
- java 1.7

## Usage
```
cd MyPaintApp
javac *.java
java MyApplication3
```
図形を描き、selectボタンで図形を選択。選択した図形をドラッグしたり、UIボタンを押すことで効果を加えられるようになる。

